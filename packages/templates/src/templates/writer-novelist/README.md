# Writer & Novelist Template

Creative writing agent for character development, plotting, worldbuilding, chapter tracking, and writing prompts.

## What It Does

- Develops detailed character profiles with motivations, flaws, and arcs
- Structures plots with acts, beats, and turning points
- Tracks chapter progress, word counts, and pacing
- Generates creative writing prompts to break through blocks
- Builds worlds: settings, magic systems, cultures, histories
- Coaches dialogue to make characters sound distinct
- Analyzes pacing to identify slow sections and rushed endings
- Supports revision with notes tracking and rewrite prioritization

## Quick Start

```bash
npm create cocapn
# Select "writer-novelist" template

cd your-brain
cocapn secret set DEEPSEEK_API_KEY
cocapn start
```

## Use Cases

- **Novelists** planning and tracking long-form fiction
- **Short story writers** developing ideas into complete narratives
- **Screenwriters** structuring scripts with beat sheets
- **Worldbuilders** creating detailed settings for RPGs or fiction
- **Writing groups** sharing character sheets and plot feedback
- **NaNoWriMo participants** tracking daily word count goals

## Configuration

Key settings in `config.yml`:
- `writing.genre`: Set primary genre for tailored suggestions
- `writing.wordCountGoal`: Project-level word count target
- `writing.dailyGoal`: Daily writing session target (words)
- `llm.temperature`: High (0.8-1.0) for creative brainstorming

## Story IP Protection

All story content, characters, and worldbuilding stay in your private repo. Your creative work is never shared externally.
