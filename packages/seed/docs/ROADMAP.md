# Roadmap

Planned milestones for `@cocapn/seed`.

## v0.2 — Connectivity

- **A2A protocol** — Agent-to-agent communication across repos
- **Multi-user sessions** — Concurrent users with isolated memory
- **Plugin system** — Loadable skill cartridges with permissions
- **Improved memory** — Confidence decay, deduplication, search

## v0.3 — Voice

- **Voice input** — Speech-to-text via Whisper or browser API
- **Voice output** — Text-to-speech responses
- **Hands-free mode** — Continuous listening with wake word

## v0.4 — Multimodal

- **Image understanding** — Describe and reason about images
- **File attachments** — Upload documents for context
- **Chart generation** — Data visualization from conversation

## v0.5 — Desktop

- **Desktop app** — Electron or Tauri wrapper for local-first use
- **System tray integration** — Background agent with notifications
- **Local file access** — Direct filesystem integration without browser sandbox

## v1.0 — Stable

- **Stable API** — Locked public interface, semantic versioning guarantee
- **Plugin marketplace** — Discover and install community plugins
- **Documentation** — Complete API reference and deployment guides
- **Production hardening** — Load testing, monitoring, error recovery
